# Bes Saiman Group — Website

Next.js 14 + Supabase + Tailwind CSS

## Запуск

### 1. Установите Node.js
Скачайте с https://nodejs.org (версия 18+)

### 2. Установите зависимости
```bash
npm install
```

### 3. Настройте Supabase
1. Создайте проект на https://supabase.com
2. Скопируйте `.env.local.example` → `.env.local`
3. Заполните ключи из Settings → API вашего проекта:
```
NEXT_PUBLIC_SUPABASE_URL=https://xxx.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=eyJ...
```

### 4. Создайте базу данных
В Supabase → SQL Editor вставьте и запустите `supabase/schema.sql`

### 5. Запустите проект
```bash
npm run dev
```
Откройте http://localhost:3000

## Структура
- `/` — Главная страница (Hero + категории + товары)
- `/catalog` — Каталог с фильтром по категориям и поиском
- `/catalog/[slug]` — Страница товара (фото, спецификации, аксессуары, калькулятор цены, скачать PDF)
- `/warehouse` — Склад (сканирование штрих-кода, приход/расход)
- `/about` — О компании
- `/contacts` — Контакты + форма обратной связи

## Языки
Переключение RU / KK / EN — кнопка в навигации (сохраняется в localStorage)

## Склад (Warehouse)
- Сканирование штрих-кода через камеру телефона/ноутбука
- Ручной ввод штрих-кода
- Приход / Расход товара с указанием количества
- История операций
- Таблица остатков

## PDF генерация
На странице товара: кнопки «Скачать тех. спецификацию» и «Скачать коммерческое предложение» — генерируются прямо в браузере без сервера.
